# Vue Weather

## Project setup

```
yarn
```

# Copy `.env.example` to `.env` and update environment variables:

cp .env.example .env

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

### Formats files with prettier

```
yarn format
```
